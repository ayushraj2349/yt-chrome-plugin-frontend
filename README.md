# 📺 YouTube Enhancement Chrome Extension

A Chrome extension that enhances the YouTube browsing experience by adding custom UI components and improving user interaction directly on the platform.

## 🚀 Features

- 🎯 Enhances YouTube UI with additional controls and overlays  
- ⚡ Lightweight and fast Chrome extension  
- 🧩 Injects custom functionality directly into YouTube pages  
- 🎥 Improves video interaction and usability  
- 🖥️ Clean and modular frontend architecture  

## 🛠️ Tech Stack

- **Frontend:** JavaScript, HTML, CSS  
- **Backend:** Flask   

## ⚙️ How It Works

- The extension injects scripts into YouTube pages  
- Custom components interact with the DOM to modify UI/UX  
- Background and content scripts handle communication and logic  
- Provides seamless integration without affecting performance
